# 🌍 Global Disaster Resilience Analysis and Visualization

An end-to-end data analytics and visualization project that analyzes global disaster events and country resilience using publicly available datasets. This project integrates socio-economic, environmental, and disaster records to identify patterns in disaster exposure, vulnerability, adaptive capacity, and recovery across countries.

The analysis includes extensive data preprocessing in Python and an interactive Tableau dashboard for exploring disaster trends, resilience indicators, and geographic patterns.

---

## 📌 Project Overview

Natural disasters have significant social and economic impacts worldwide. This project combines multiple global datasets to evaluate disaster resilience and understand how socio-economic and environmental factors influence a country's ability to prepare for, respond to, and recover from disasters.

The project includes:

- Data cleaning and integration from multiple sources
- Feature engineering and resilience index calculation
- Exploratory data analysis
- Interactive Tableau dashboard
- Geographic, temporal, and comparative visualizations

---

## 📊 Dashboard Features

The Tableau dashboard includes interactive visualizations such as:

- 🌍 Choropleth maps showing country resilience scores
- 📈 Time-series analysis of disaster trends
- 🫧 Bubble charts comparing vulnerability and adaptive capacity
- 🌳 Treemaps of disaster type distribution
- 📉 GDP vs Recovery analysis
- 🌐 Geographic disaster impact analysis
- 🎯 Interactive filters for:
  - Country
  - Year
  - Disaster Type
  - Disaster Group

The project integrates data from:

- UNDP Human Development Index (HDI)
- EM-DAT International Disaster Database
- World Bank Development Indicators
- World Bank Governance & Financial Indicators

## 📊 Key Metrics Developed

Several custom indicators were engineered, including:

- Country Resilience Index (CRI)
- Adaptive Capacity Index
- Exposure Index
- Vulnerability Index
- Disaster Impact Index (DII)
- Recovery Rate Proxy
- Climate Risk Proxy
- Infrastructure Exposure
- Human Cost Ratio
- Fatalities per Million
- Affected Population Percentage

---

## 📈 Key Insights

- Developed countries generally demonstrate stronger disaster resilience and recovery capacity.
- Floods and storms are the most common disaster types globally.
- Disaster frequency has increased over the past two decades.
- Economic development improves recovery outcomes but is not the sole determinant of resilience.
- Regional differences highlight varying levels of disaster preparedness and adaptive capacity.
